The fossilisation of complex, mainframe based Enterprise IT environments is a major and well documented problem. Moreover it is incredibly expensive, both in terms of maintenance and change prohibition. The IT environments of large corporations, are incredibly complex, often involving thousands of systems and millions of lines of code. This problem is compounded by the fact that the skill base required to maintain and modify these systems is shrinking as those with the requisite skills retire.

The complexity and lack of skills makes the complex IT environment incredibly hard to map, using traditional means, and is prone to error. These two factors make the mapping process of the IT environment prohibitively expensive. Without an IT environmental map, application migration is difficult if not impossible.



Eveve has produced a truly revolutionary solution to the problem of mapping complex IT environments. Mercator, a unique Eveve proprietory technology, breaks down all elements in a corporate IT environment to machine code level. It reconstructs the environment using this machine code base, and maps every linkage between every element in the IT environment.

Mercator enables:

Application portfolio management: understand the complex linkages between your different IT systems with complete accuracy. Studies have shown that Mercator can provide an 80% cost saving on the analysis of an IT environment.

Accurate mapping: Mercator provides a 100% accurate map of your IT environment, impossible using traditional methods. This is essential for the impact analysis of change.

Facilitation of platform migration: by understanding your environment you can migrate your IT systems with the knowledge of which systems are affected. This is essential if you want to upgrade older systems, or merge systems.
