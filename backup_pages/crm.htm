The Eveve Client Relationship Management (CRM) System is the ultimate modern live booking system. CRM combines all the functionality of live table management and structured online bookings with an integrated database of the restaurant's customers.

When bookings arrive in the system they are cross referenced with customer entries in the database. If the person has eaten in the restaurant before, the system will recall their previous behaviour and preferences. For example the system will highlight that the person who has just booked is a VIP, friend of the restaurateur, blacklist etc. If required the system can also detail the number of times the customer has eaten there before, together with average spend and dining preferences. If the person is not recognised, their details will be incremented to the database, creating an ever expanding management tool.

As well as reactive management information, the system is capable of proactive marketing campaigns, if necessary. Eveve marketing has two elements which set it apart from the traditional methods. First, the target audience can be refined by searching on variables in the customer database, for example, local wine conoisseurs whose average spend exceeds £50.

Second, the target audience can be reached within seconds using Eveve Text and Email. Every one of these can be custom made, using variables from the customer database in order to give the personal touch. For example, a text would start with "Dear Tony" rather than "Dear Sir".

CRM integrates with our EPOS tills, so that databases on customers and spending habits can be integrated. Not only can you see how regularly someone is dining but also monitor their average spend, and see which items have been purchased, improving customer service and management information.
